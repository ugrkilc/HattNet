# HattSet-12 Dataset

**HattSet-12** is a publicly available, expert-annotated dataset for **Arabic calligraphy style classification**, containing images across **twelve major calligraphic scripts**.

## 📥 Download

The dataset is hosted on Kaggle:

**➡️ [HattSet-12 on Kaggle](https://www.kaggle.com/datasets/ugrkilc/hattset-12)**
